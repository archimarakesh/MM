# Guard-бот (капча-ознакомление)

Отдельный бот и отдельный сервис Railway — новый участник чата обязан нажать
«Ознакомлен с правилами» в течение 5 минут, иначе автокик. Не нагружает магазинного бота.

## Настройка

1. **Создай нового бота** в [@BotFather](https://t.me/BotFather) → `/newbot`, получи токен.
2. Добавь бота **администратором** в чат покупателей с правами:
   - Ограничивать участников (Restrict members)
   - Банить участников (Ban users)
   - Удалять сообщения (Delete messages)
3. На Railway: **New Service → Deploy from GitHub repo** (тот же репо), в настройках сервиса
   укажи **Root Directory = `guard`**. Переменные:
   - `GUARD_BOT_TOKEN` — токен нового бота (обязательно)
   - `RULES_CHAT_ID` — ID чата (узнать: добавь бота, отправь `/chatid` в чате). Без него гейт работает во всех чатах, где бот админ.
   - `RULES_TIMEOUT` — секунд на ознакомление (по умолчанию 300).

Текст правил — в `bot.py`, константа `RULES_TEXT`.
